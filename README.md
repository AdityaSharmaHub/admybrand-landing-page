### Prerequisites
- Node.js 18+

### Installation

1. Clone the repository
```bash
git clone https://github.com/AdityaSharmaHub/admybrand-landing-page.git
cd admybrand-landing-page
```

2. Install dependencies
```bash
npm install
# or

```

3. Run the development server

```bash
npm run build
npm start
```
